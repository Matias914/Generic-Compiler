## Identificadores:
    Se prueba los siguientes casos posibles y válidos:
    - Variable compuesta de letras mayúsculas
    - Variable compuesta de letras y números
    - Variable compuesta de letras y porcentajes
    - Variable compuesta de letras números y porcentajes
    - Variables de cantidad de caracteres extendidas

## Errores Léxicos:
    Si bien no pueden tener errores léxicos sí que puede
    ser confusa la tokenización. Se proponen los siguientes casos:
    - Entero Sin Signo + Variable
    - Caracter Inesperado + Variable
    - Variable + Palabra Reservada
    - Variable "." Variable
